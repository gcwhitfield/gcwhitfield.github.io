Slope Crusher

Spring 2020 - Shader & Gameplay Progamming (Unreal)

Overview

Slope Crusher is an intense racing game about boarding through a noise-filled world. Jump, Smash and Flip at insane speeds to get the best time possible.

This is the fourth project I did with the Game Creation Society. I met with other students once a week for half of the semester, and then transitioned to virtual calls via Discord once the pandemic started.

This was the first project I've ever worked on using Unreal Engine 4. I created the emissive noise shaders and destruction blueprint for the destructible objects in the scene. I also creaeted a ferrofluid-esque water shader, and configured the audio groups for the options screen. There was a pretty big learning curve at the start since I didn't know how to use Unreal's blueprint system. I watched tons of online tutorials and ended up with some pretty cool visuals! Check out Slope Crusher's page on itch.io to download the game for Windows/MacOS.
