Game Bytes

Fall 2019 - Gameplay Progamming (Unity)

Overview

Game Bytes is a minigame collection created for the Game Creation Society cabinet (arcade machine). Game Bytes currently features 4 minigames: Marble Roller: Roll your marble to the end as fast as you can! Sporshmallow: Deal as much damage as you can with sports themed characters! Shotpot: A physics based game where you try to grab as much food as you can out of a hotpot! Tetris 2P: A tetris 1v1 where you defeat the other player by clearing grey lines!
