Space Tanks

Summer 2020 - Programming (Unity), Music & Sound Effects

Overview

Space Tanks is a two player turn-based game where players shoot each other as tanks on the moon. I created this within 48 hours and submitted it to United Game Jam 2020.

The theme of the jam was "Two", and I decided to incorporate it by making the game two player. Since it was my first game jam, I wanted to keep it simple. Each turn, players have two options: move or shoot. The background is a low-poly moon landscape created with Unity's terrain package and the user interface (UI) was created with Unity default UI assets.

A lot of my time working on Space Tanks was spent debugging in Tank.cs and creating the moon terrain. If I were to do this again, I would spend much less time creating the level environment and more time playtesting. As the jam deadline came closer, I found myself rushing more and more, leaving little time for playtesting. As a result, the aiming feel pretty clunky. That being said, I am still very happy with what I was able to do in such a short time.

Also, I discovered that Unity will not always do what I expect in build vs in editor. I spent a long time debugging an issue where projectile collisions only ran in the Unity editor but not in the build, which caused the game to be un-playable. As it turns out, gameObject.isStatic might not always return the right thing inside of OnTriggerEnter!

Feel free to check out Space Tanks on itch.io!
